# Resume Website

A personal resume website for Shikhar Parashar, showcasing professional experience, skills, projects, and education.

## Overview

This is a clean, HTML-based resume website that presents professional information in a structured format. The resume highlights expertise in AI development, full-stack development, web design, and content creation.

## Features

- **Professional Sections:**
  - About Me
  - Skills
  - Work Experience
  - Projects
  - Education
  - Contact Information

- **Technologies Highlighted:**
  - Frontend: HTML5, CSS3, JavaScript
  - Frameworks: React, Vue.js
  - Backend: Node.js, Express
  - Databases: MongoDB, MySQL
  - Other: Git, GitHub, AI Development

## Usage

1. **View the Resume:**
   - Simply open `resume.html` in any modern web browser
   - No additional setup or dependencies required

2. **Customize:**
   - Edit `resume.html` directly to update information
   - Modify content, add sections, or update styling as needed
